    Aria - Adaptive Reasoning & Intelligence Assistant

  A system prompt engineering project by: Simplefanatic (https://github.com/odieo1)

-----

What is Aria?

    A.R.I.A. (Adaptive Reasoning & Intelligence Assistant) is a carefully crafted AI system prompt designed to produce a conversational assistant that is context-aware, dynamically responsive, and capable of adapting its reasoning style to the needs of the user.

    Rather than a rigid, one-size-fits-all assistant, A.R.I.A. is built around the idea that good AI interaction should feel fluid — shifting tone, depth, and approach based on context, without losing coherence or personality.

-----

   Goals

- Deliver consistent, intelligent responses across a wide range of topics and conversation types
- Adapt communication style to casual, technical, creative, and emotional contexts
- Maintain a defined personality without feeling robotic or scripted
- Handle ambiguous or complex prompts gracefully
- Resist prompt confusion, contradiction loops, and edge-case breakdowns

-----

 🧪 Stress Testing/Community Help Wanted

This prompt is currently in active development and I'm requesting community stress testing.

If you'd like to help, try throwing A.R.I.A. at:

    Edge cases ambiguous questions, contradictory instructions, nonsense inputs
    Adversarial prompts - attempts to break persona, jailbreaks, role confusion
    Multi-turn coherence - long conversations where context should carry forward
    Domain depth - technical, creative, emotional, philosophical - how does she hold up?
    Tone shifts - start casual, go technical mid-conversation, go emotional - does she adapt smoothly?

    How to Submit Findings

Open an issue(../Aria-2.2/issues) and include:

- The platform/model you tested on (e.g. OpenRouter + LLaMA 3.3 70B)
- The prompt or exchange that triggered the behavior
- What you expected vs. what happened
- Any suggested fix if you have one

All feedback is welcome, breakdowns are just as valuable

    Usage

Copy the prompt from the relevant file in this repo and paste it as the system prompt in your preferred AI interface or API call.

Compatible with any instruction-following model

    Tested on:
- LLaMA 3.3 70B (via OpenRouter)
- Google Gemini 2.0/2.5 Flash
- Other OpenAI-compatible endpoints
